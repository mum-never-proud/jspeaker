How to use jSpeaker

Want to make your websites speak?

jSpeaker is intended to make your website more interactable!

Getting started

Before calling any jSpeaker methods make sure it is initialized either by placing the class in HTML or by calling the jSpeaker.init() method

"js-jspeaker" class

Place this class anywhere in the HTML Page (<body> recommended) and stay calm!

          
            ...
            <body class="js-jspeaker">
              ...
            </body>
            ...

Lazyload

You can also initialize jSpeaker only after an event has occured or after certain amount of time

          
            jSpeaker.init(optional_voice); // initialize jSpeaker with an optional voice or default system voice

Control

You can turn jSpeaker on / off at any point!

          
            jSpeaker.off(); // website won't speak anymore on text selection
            jSpeaker.on(); // website starts speaking again

Speak

jSpeaker can also speak any given string!

          
            jSpeaker.speak('Hello World'); // speaks the given string

Cancel

jSpeaker cancels speaking on deselecting the text, however you can also explicity cancel the speaking anytime using jSpeaker.cancel()

          
            jSpeaker.cancel(); // cancel the speaking
